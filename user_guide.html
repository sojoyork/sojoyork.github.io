What is cookie monster?

cookie monster is a tool made for fetching website cookies if ever needed or someday needed. You can use it for checking if a cookie is secure. I devloper got the idea of cookie monster while eating 1 cookie

Let's use cookie monster

So first open file exploerer (or any other file manager that you use in your OS) then open cookie_monster.folder then open cookie_monster.py and then a window will open which will ask you for a website URL to fetch cookies of so let's choose http://facebook.com (because it has the most cookies and also i chose facebook for educatenal proposes) and now type http://facebook.com and press enter then cookie monster will ask you to conform to fetch the cookies of the website by typing fetch so type fetch and it will make a file on your computer and open cookie_monster.folder and you will see a text file named facebook.com.Fetched.string.txt (mostly .txt is not present in the name so dont het consused) and open the file and CONGRATULATIONS! You fetched a cookie! (you can also use the cookie_search.py to read what is in the cookie)

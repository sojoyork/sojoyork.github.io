User Guide for Cookie Monster

What is cookie monster?

cookie monster is a tool made for fetching website cookies if ever needed or someday needed. You can use it for checking if a cookie is secure. I devloper got the idea of cookie monster while eating 1 cookie

Let's use cookie monster

NOTE: make sure that you have cookie monster installed

Step 1. Open file explorer or any other file manager that you use in your OS
step 2. open cookie_monster.folder

step 3. open cookie_monster.py

step 4. type the URL of a website when prompted e.g http://example.com

step 5. Cookie_monster.py will save the information inside a text file.

step 6. open the text file and CONGRATULATIONS! you fetched a cookie(s)
